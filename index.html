Features

Everything Your Graphs Need

Gridline Blueprints is a comprehensive suite of editor tools designed to make Blueprint graphs readable, organized, and beautiful.

⊞

Orthogonal Wire Routing

Replaces default spline wires with clean right-angle paths. Configurable pin length, routing bias, and U-turn clearance keep every connection tidy.

⫼

Wire Spacing

Overlapping parallel segments are automatically spread apart so individual wires stay distinct — even in dense graphs.

⬡

Node Avoidance

Wires intelligently route around unrelated nodes. Choose between simple sideways shifts or complex multi-segment detours.

⊕

Junctions

Where wires cross, Gridline draws clear visual indicators — Break gaps, Jump arcs, Ring outlines, or filled Discs — so overlaps are never ambiguous.

◎

Flow Animation

Animated dots or pulses travel along wires to indicate data-flow direction. Filter by execution wires, data wires, selected nodes, or all wires.

⊟

Hidden Reroute Nodes

Reroute nodes become invisible along the wire for a cleaner look. They reappear instantly when hovered for easy editing.

⬖

Named Reroute Nodes

Declaration & Usage node pairs let you name and reference values across graphs — eliminating long-distance wires and reducing clutter.

⊞

Node Alignment

One keypress snaps selected nodes into a clean, grid-aligned layout. Execution chains, data inputs, and branches are recursively organized.

☰

Context Menu Enhancements

Fuzzy search with typo tolerance, auto-expanding categories, resizable menus, and a recently-used nodes section with keyboard cycling.

Showcase

How It Works

See how Gridline transforms Blueprint graphs from tangled webs into clear, structured diagrams.

Orthogonal Routing with Wire Spacing

Wires follow strict horizontal and vertical paths with configurable pin stubs. Parallel segments are automatically spaced apart for clarity.

Node Avoidance & Junctions

Wires automatically detour around unrelated nodes. Where wires cross, junction indicators (jumps, rings, discs, or breaks) provide unambiguous visual clarity.

Named Reroute Nodes

Declare a named value once, then reference it anywhere with lightweight Usage nodes. Eliminates long-distance wires and can be converted back to standard reroutes at any time.

One-Press Node Alignment

Select nodes and press a single key. Execution chains, data dependencies, and branches are recursively organized onto a clean grid with optional wire straightening.

Configuration

Fully Configurable

Every aspect of Gridline Blueprints is exposed through Editor Settings — no code changes or config file editing required.

Wire Routing

  • Source / Destination routing bias
  • Adjustable pin length (12–64 px)
  • U-turn clearance style
  • Grid snapping for U-turn bridges
  • Short wire threshold & style

Appearance

  • Wire thickness multiplier
  • Corner size & style (Eased / Fillet / Chamfer)
  • Tight turn handling & threshold
  • Hidden reroute node toggle

Wire Spacing

  • Spacing distance (8–64 px)
  • Shared source wire spacing
  • Shared destination wire spacing

Node Avoidance

  • Simple / Complex avoidance modes
  • Avoidance padding (0–64 px)

Junctions

  • Break / Jump / Ring / Disc styles
  • Junction size (8–32 px)

Flow Animation

  • Dot / Pulse animation styles
  • Wire type filter (Exec / Data / All / Selected)
  • Speed, spacing, size & brightness

Node Alignment

  • Configurable keybindings
  • Selected-only alignment mode
  • Execution wire straightening
  • Group alignment mode

Context Menu

  • Fuzzy search with typo tolerance
  • Auto-expand categories
  • Custom menu width & height
  • Recent nodes with cycle keys
